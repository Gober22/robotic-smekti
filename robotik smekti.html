Tentang Kami

Ekstrakurikuler Robotika SMEKTI

Robotik SMEKTI adalah wadah bagi siswa-siswi yang memiliki minat dan bakat dalam bidang robotika dan teknologi.

Kami fokus pada pengembangan keterampilan teknis, kreativitas, dan kemampuan problem solving melalui berbagai project dan kompetisi robotika tingkat regional hingga nasional.

Visi Kami

Menjadi ekstrakurikuler robotika terdepan yang melahirkan inovator-inovator muda di bidang teknologi.

Misi Kami

  • Mengembangkan kemampuan teknis siswa dalam bidang robotika
  • Mendorong kreativitas dan inovasi melalui project-based learning
  • Mempersiapkan siswa untuk kompetisi robotika tingkat nasional
  • Membangun karakter dan teamwork melalui kolaborasi tim

Kompetisi Kami

LKS Mobile Robotik

Kompetisi robotika berbasis mobile yang menguji kemampuan programming dan kontrol robot menggunakan smartphone.

  • Programming Android
  • Sensor Integration
  • Wireless Control

Sumo Robot

Pertarungan robot satu lawan satu untuk mendorong lawan keluar dari arena. Menguji strategi dan kekuatan robot.

  • Mechanical Design
  • Sensor Strategy
  • Power Management

Line Follower

Robot otomatis yang mengikuti jalur dengan kecepatan dan akurasi tinggi. Klasik namun menantang!

  • PID Control
  • Sensor Calibration
  • Speed Optimization

Ultrasonic Challenge

Navigasi robot menggunakan sensor ultrasonik untuk menghindari rintangan dan mencapai target.

  • Distance Measurement
  • Obstacle Avoidance
  • Path Planning

Fire Fighting Robot

Robot pemadam api otomatis yang mendeteksi dan memadamkan api dalam maze.

  • Flame Detection
  • Navigation Algorithm
  • Extinguishing System

Transporter Robot

Robot pengangkut barang otomatis dengan kemampuan sorting dan delivery.

  • Object Recognition
  • Gripper Design
  • Route Optimization
